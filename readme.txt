This widget retrieves your list of tracks you 'love' on The Hype Machine ( hypem.com ) as well as other HypeM track listings.

== Installation ==

1. Upload `widget-hypemachine.php` to the `/wp-content/plugins/widgets` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the presentation -> widgets section and ad your own title, your Hypem.com username and what you want the widget to display
